# 🚀 TaskNova — Smart Task Management System

**TaskNova** is a modern and lightweight task management web application designed to help users organize daily tasks efficiently. It provides a clean UI, smooth interactions, and productivity-focused features built using **HTML, CSS, and JavaScript**.

---

## 📌 Overview

TaskNova helps users plan, track, and manage their tasks in a structured way. The application focuses on simplicity, performance, and usability while delivering essential productivity tools for students and professionals.

The system allows users to create, manage, and monitor tasks while maintaining an intuitive and responsive experience across devices.

---

## ✨ Features

* ✅ Add, edit, and delete tasks
* 📅 Task organization and tracking
* 🔍 Clean and modern user interface
* 💾 Local storage support (data saved in browser)
* 🔐 Login & logout functionality
* ⚡ Smooth scrolling and responsive design
* 🎯 Productivity-focused layout
* 📱 Mobile-friendly UI

---

| Technology                  | Purpose                       |

## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/rashigarg04/tasknova.git
   ```

2. Navigate to the project folder:

   ```bash
   cd tasknova
   ```

3. Open `index.html` in your browser.

No additional dependencies or installations are required.

---

## 🚀 Usage

1. Open the application in your browser.
2. Log in to access the dashboard.
3. Add new tasks using the task input section.
4. Manage tasks by editing or deleting them.
5. Tasks are automatically saved using browser storage.
